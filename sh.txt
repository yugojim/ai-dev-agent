✅ Step 0 — 進入 WSL / Ubuntu
wsl

確認 Python：

python3 --version

若沒有：

sudo apt update
sudo apt install -y python3 python3-venv python3-pip git curl unzip
✅ Step 1 — 建立 Agent Workspace
mkdir -p ~/ai-dev-agent
cd ~/ai-dev-agent
✅ Step 2 — 建立 Python 虛擬環境
python3 -m venv venv

啟動：

source venv/bin/activate

看到：

(venv)

代表成功。

✅ Step 3 — 升級 pip
pip install --upgrade pip wheel setuptools
✅ Step 4 — 安裝 Agent 必備 Python 套件

👉 這一組是 Full Agent Runtime 必備

pip install \
requests \
python-dotenv \
rich \
tenacity \
playwright \
pydantic \
gitpython \
tqdm \
pyyaml \
✅ Step 5 — 安裝 Playwright Browser（非常重要）

只 pip install 還不夠
必須下載 browser runtime：

playwright install

如果你要 headless chrome 穩定：

playwright install chromium
✅ Step 6 — 安裝 Node（給某些 Cognito / UI flow）
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

確認：

node -v
npm -v
✅ Step 7 — 安裝 Codex CLI（核心）
⭐ 建議方式（最穩）
npm install -g @openai/codex

確認：

codex --version
✅ Step 8 — 設定 OpenAI API Key
nano ~/.bashrc

加入：

export OPENAI_API_KEY="sk-xxxxx"

套用：

source ~/.bashrc

測試：

codex exec "say hello"
✅ Step 9 — Git SSH（Agent 自動 clone 必備）
ssh-keygen -t ed25519
cat ~/.ssh/id_ed25519.pub

貼到 GitHub / GitLab。

測試：

ssh -T git@github.com
✅ Step 10 — Agent 需要 Maven / Java
sudo apt install -y openjdk-21-jdk maven

確認：

java -version
mvn -version
✅ Step 11 — 測試 Spring Boot 可跑
cd ~/ai-workspaces/issue-XXXX/repo
mvn clean package -DskipTests --quiet
mvn spring-boot:run
✅ Step 12 — Agent 執行模式
⭐ 只跑 Codex

（現在最建議）

python agent_loop.py --mode codex_only
⭐ Codex 完 → 手動驗證
cd ~/ai-workspaces/issue-XXXX/repo
mvn clean package
mvn spring-boot:run
⭐ 自動跑 Build Phase
python agent_loop.py --mode build_only
⭐ Full Autonomous
python agent_loop.py --mode full

流程：

Codex → Build → Runtime → Auth → Playwright → Report
✅ Step 13 — 網頁登入自動化（Cognito）

Agent 若要做：

自動登入 → 保存 session → 跑 UI smoke test

需要：

Python side

已裝：

playwright
requests
Browser runtime
playwright install
